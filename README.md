# 魔王のダイスカジノ（saikoro-bj）

サイコロを使ったミニゲーム集（1アプリ）。ホームから各ゲームを選んで遊ぶ。

- **サイコロBJ** … 合計を目標数(1〜18)に近づけて魔王と勝負（引き際の駆け引き）
- **サイコロプロスピ** … 魔王の投球(3個振り)の出目を予想して打つダイス野球
- **スロット 555 / 777** … 数字を揃えて❤️を獲得（役・確変あり）

共通通貨: 🪙コイン（BET・配当） / ❤️ハート（スロット特典）

## 技術スタック（すべて無料枠）
バニラJS / HTML+CSS（モバイルファースト） / localStorage / GitHub Pages / 音声は事前生成mp3。
詳細は `docs/設計書.md` を参照。

## ディレクトリ構成
```
saikoro-bj/
├─ README.md
├─ docs/
│   ├─ 設計書.md         … これ1枚で開発に入れる完全設計書
│   └─ 画像プロンプト.md  … 必要画像11枚の生成プロンプト＋サイズ指定
├─ assets/
│   ├─ images/           … ゲームが読み込む本番アセット（軽量WebP・計約3.9MB）
│   │   └─ README.md     … 収録ファイル一覧＋透過の宿題
│   ├─ source/           … 原本の高解像度PNG（再編集・背景除去用・計約26MB）
│   └─ reference/        … 参考用（パッケージ絵・UI/UXモック）
│       ├─ package.png
│       └─ uiux_mock.png … この見た目を目標に実装する
└─ （src/ などの実装は開発セッションで追加）
```

> 画像はWebPで軽量化済み（背景2MB→約150KB等）。実装は `background-size: cover` で読み込む。

## 開発の始め方（新規セッション）
